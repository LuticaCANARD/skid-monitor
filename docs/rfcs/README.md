# RFCs

Skid Monitor의 구조적 결정과 운영 계약을 기록한다.

## Umbrella Rule

`skid-monitor`는 SKID 계열 실험을 합치는 canonical integration repository다. 다른 `skid-*`
repository는 조사와 설계의 source로만 취급하고, 채택할 기능은 먼저 이 RFC 목록에 기록한다.
실제 구현은 이 repository 안에서만 진행한다.
